# Team India T20I Stats Analysis

This project analyzes Team India's Men's T20 International performance using ball-by-ball data to identify the fastest team milestones (50, 100, 150, 200, 250 runs).

## Files Included

1.  **dashboard.html**: An interactive dashboard to visualize the fastest milestones. Open this in any web browser.
2.  **dashboard_data.js**: Data file required for the dashboard to function. Do not delete or move this.
3.  **team_india_milestones.csv**: A CSV file containing the computed list of fastest milestones.
4.  **india_ball_by_ball.csv**: The raw ball-by-ball data for all Indian Men's T20I matches used in this analysis.

## How to View the Dashboard
Simply double-click `dashboard.html`. It will open in your default web browser. Note: It requires an internet connection to load the charting library.

## Source Code
- `process_data.py`: Python script used to process the raw data.
- `download_data.py`: Script used to download data from Cricsheet.
